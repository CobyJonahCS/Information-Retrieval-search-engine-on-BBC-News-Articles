# Information-Retrieval-search-engine-on-BBC-News-Articles
Information Retrieval Search Engines to Query BBC News articles

## Dataset

This project uses the [BBC Articles Dataset](https://www.kaggle.com/datasets/jacopoferretti/bbc-articles-dataset).

## Setup

Clone the repository and move into the project folder:

```bash
git clone <repo>
cd <repository-folder>

conda create --name <env_name> python=3.12
conda activate <env_name> 

pip install -r requirements.txt

cd frontend
python app.py

```

```markdown

After starting up the Flask service, navigate to http://127.0.0.1:5000 in your browser.
```
